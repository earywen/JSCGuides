Belo'ren, Child of Al'ar

🛡️ Tanks

  • Guardian's Edict : Combo frontaux Light/Void. Chaque tank soak sa couleur. Le 3ème frontal est les deux en même temps.
  • Tank Swap : 5 attaques de mêlée après les frontaux font des dégâts massifs. Swap après chaque série.
  • Feather : Identifiez votre couleur (Light/Void). Ne soakez QUE votre frontal correspondant.

💚 Healers

  • Eternal Burns : Absorbe les soins. Spot heal les joueurs ciblés pour retirer l'absorb.
  • Phase 2 : Fort DoT raid pendant l'intermission. Préparez vos CDs, surtout ceux utilisables en mouvement.
  • Mêlées Tank : Après les frontaux, le tank actif prend 5 hits très lourds. Soins ciblés urgents.

⚔️ DPS

  • Ember Adds : Tuez les Ember Phoenix PUIS leurs œufs avant qu'ils ne respawn. Interrompez Eruption (votre couleur uniquement).
  • Bloodlust : Sur l'œuf en Phase 2. Les dégâts sur l'œuf sont persistants entre les intermissions.
  • Radiant Echoes : Créez un gap en absorbant 2-3 orbes de votre couleur, puis le raid passe à travers.

Tactique Globale

  • Bloodlust : Sur l'œuf en Phase 2. C'est la phase de burn.
  • Convergence : Au pull, le raid est divisé en 2 équipes (Light/Void). Interagissez toujours avec VOTRE couleur, évitez l'autre.
  • Positionnement : Boss au centre. Les soaks Embers se font sur les côtés. Stacker les cercles est possible pour faciliter le cleave.
  • Dégâts Œuf : Les dégâts infligés à l'œuf sont permanents. Chaque rebirth applique -10% soins sur le raid.

Phase 1 : Phoenix Reborn

Belo'ren combat sous forme de Phoenix. Le raid est divisé en équipes Light et Void via Voidlight Convergence.

Mécanique Effet Action Requise
Voidlight Convergence Au pull, dégâts raid + assigne Light Feather ou Void Feather. Light réduit les dégâts Holy de 50%, Void réduit les dégâts Shadow de 50%. Identifiez votre couleur. Interagissez UNIQUEMENT avec les mécaniques de votre couleur.
Embers of Belo'ren Cercle sur 1-2 joueurs (un Light, un Void). Après quelques secondes : dégâts massifs + knockback. Laisse une flaque et fait spawn un Ember Phoenix. Soak : Les joueurs de la même couleur stackent pour split. Ou stackez tout derrière le boss pour cleave les 2 adds.
Ember Phoenix Light/Void Eruption : Cast qui wipe si non interrompu. Light/Void Blast : AoE interruptible.
À la mort, laisse un œuf qui essaie de respawn. 		Interrupt Eruption (par un joueur de la même couleur). Tuez l'add puis l'œuf avant le respawn.
Infused Quill Flèche Light ou Void tirée sur un joueur. Touche le premier joueur sur le trajet. La cible s'arrête. Un joueur de la couleur opposée intercepte la flèche en se plaçant entre le boss et la cible.
Radiant Echoes Ligne d'orbes Light et Void traversant la salle. Contact avec une orbe de la mauvaise couleur = mort. 2-3 joueurs absorbent les orbes de leur couleur pour créer un passage, puis tout le raid passe.
Guardian's Edict Combo de 3 frontaux : Light Edict, Void Edict, puis Voidlight Edict. Chaque frontal fait de lourds dégâts magiques + physiques. Si raté, +30% dégâts du boss pendant 30s. Tank 1 soak son frontal couleur → Tank 2 taunt + soak son frontal couleur → Les deux soakent le 3ème (un de chaque côté).
Eternal Burns Absorbe de soins sur joueurs aléatoires. Inflige des dégâts tant que l'absorb n'est pas retiré. Healers : Spot heal les cibles pour retirer l'absorb rapidement.

Phase 2 : Ashen Shell (Intermission)

À 0% PV, Belo'ren se transforme en œuf et cast Rebirth. Les dégâts sur l'œuf sont permanents. Quand le cast de Rebirth termine, la Phase 1 recommence.

Mécanique Effet Action Requise
Immortal Flame / Rebirth L'œuf commence à cast Rebirth. Les dégâts infligés à l'œuf persistent entre chaque intermission. Le combat se termine quand l'œuf meurt. Bloodlust ici. DPS maximum sur l'œuf. Les dégâts se cumulent entre les phases.
Incubation of Flames Le sol se divise en sections Light et Void qui explosent alternativement. Restez sur les sections de votre couleur pour chaque explosion. Bougez entre les slices.
Ashen Benediction Chaque Rebirth réussi applique -10% soins reçus sur tout le raid (stack cumulatif). Soft enrage. Tuez l'œuf le plus vite possible pour limiter les stacks.
Eternal Burns (P2) Absorbe de soins sur joueurs aléatoires pendant la phase d'œuf. Spot heal les cibles. Coordonnez les CDs de heal avec le mouvement.

Mythique Only

Mécanique Effet Action Requise
Infused Quill (Mythique) L'intercepteur prend une AoE 8m. Toucher d'autres joueurs lors du soak est dangereux. L'intercepteur doit s'éloigner de la cible après avoir soak.
Radiant Echoes (Mythique) Si une orbe touche le boss, elle explose pour des dégâts raid massifs. Positionez le boss pour qu'il ne soit PAS dans le chemin des orbes.
