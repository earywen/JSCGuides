Salhadaar

🛡️ Tanks

  • Shattering Twilight : L'impact fait apparaître des pics. Éloignez-vous du raid AVANT l'impact.
  • Kiting : Déplacez le boss loin des orbes de Concentrated Void pour donner du temps aux DPS.
  • Instability : Swap à chaque Shattering Twilight ou quand le dot devient trop lourd.

💚 Healers

  • Cosmic Unraveling : Phase de burn (20s). Soins de zone massifs requis pendant les rayons.
  • Despotic Command : Dispelez les joueurs UNIQUEMENT lorsqu'ils sont au bord de la salle.
  • Heal Absorb : Soignez intensivement le joueur après le dispel pour vider l'absorb.
  • Orb DOT (Héroïques) : Ne tuez pas les orbes en même temps ! Laissez le DOT expirer.

⚔️ DPS

  • Focus Orbes (Héro) : Tuez les orbes UNE PAR UNE. Ne les tuez jamais simultanément.
  • Bloodlust : Utilisez la BL pendant l'Unbound Phase (+25% dégâts subis par le boss).
  • Despotic Command : Si vous avez le cercle rose, courez IMMÉDIATEMENT au bord de la salle.
  • Fractured Images : Priorité ABSOLUE aux interruptions pour éviter de remplir la salle de vide.

Tactique Globale

  • Bloodlust : Phase Unbound. Le boss subit +25% de dégâts, c'est le moment d'envoyer tout le burst.
  • Gestion des Orbes : Ne tuez jamais plusieurs orbes simultanément (Héro). Attendez que le DOT raid expire.
  • Zones de Vide : Si vous avez un cercle (Despotic Command), posez-le sur les bords pour garder le centre propre.
  • Positionnement : Tanks, orientez les Shattering Twilight vers les bords pour ne pas encombrer le centre.

Le Vide Déchaîné : Unbound

Salhadaar gagne de l'énergie passivement. À 100 Énergie, il entre en **Cosmic Unraveling**.

Phase Description Stratégie
Phase Normale Attire des orbes de Concentrated Void vers lui depuis les bords. Interceptez/Tuez (Une par une en Héroïque). Priorité aux orbes les plus proches.
Unbound Phase À 100 énergie, le boss subit +25% dégâts pendant 20 sec. Phase de Burst (BL) : Évitez les rayons rotatifs (Uncontainable Cosmos).

Sorts et Dangers

Capacité Effet Contre-mesure
Shattering Twilight Huerle une étoile noire + pointes au sol (Twilight Spikes). Sortez vite des zones d'impact et des lignes de pointes.
Uncontainable Cosmos Rayons d'énergie cosmique partant du boss dans plusieurs directions. Évitez le contact direct avec les rayons.
Despotic Command Cercle rose sur un joueur. Laisse une zone de vide + heal absorb au dispel. Bords de salle : Courez au bord avant le dispel. Healers, videz l'absorb.
Twisting Obscurity Éclairs de ténèbres qui sautent entre tous les joueurs. Restez packés pour faciliter les soins de zone.
Echoing Star La Shattering Twilight rebondit sur plusieurs joueurs. Dispersion maximale lors de l'incantation.
Fracturer Projection Le boss se divise en images qui incantent Unstable Nexus. Interrompez les images. Brisez les Nexus Shields en priorité.
